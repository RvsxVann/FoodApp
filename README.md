Food App is a web application that allows users to search for meals and explore detailed recipes using data from TheMealDB API.

## Features

* Search meals by name
* View meal images and information
* Display ingredients and measurements
* Read detailed cooking instructions
* Responsive and user-friendly interface

## Technologies Used

* HTML5
* CSS3
* JavaScript (ES6)
* Bootstrap 5
* TheMealDB API
* Font Awesome

## Preview

Users can search for their favorite meals and view complete recipe details in a modal window.
